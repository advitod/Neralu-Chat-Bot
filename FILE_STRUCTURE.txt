WHAT EACH FILE DOES
==================================================

app.py:
  - Runs the chatbot server
  - Handles user messages
  - Detects intent (greeting, questions, etc.)
  - Generates responses
  - Captures leads (name + phone)
  - Stores conversation history
  - Provides API endpoints

templates/index.html:
  - Chat interface users see
  - Message display area
  - Input box and send button
  - Quick action buttons
  - Styling and animations
  - JavaScript for interaction

advanced_chatbot.py:
  - Alternative to app.py
  - Uses OpenAI GPT API
  - More intelligent responses
  - Better natural language
  - Requires API key and setup

requirements.txt:
  - Lists Python packages needed
  - Used by: pip install -r requirements.txt
  - Contains: flask, flask-cors, gunicorn

.env.example:
  - Template for environment variables
  - Copy to .env and fill in
  - For API keys, database URLs, etc.
